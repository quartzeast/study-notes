a = b+=c++-d+--e/-f 怎么理解？

    a = b += c++ - d + --e / -f
    因为变量只引用一次，所以++或--在后面的对式子结果不会有影响
    式子变为 a = b += c - d + --e / -f
    自增自减可以分别改为+1、-1
    式子变为 a = b += c - d + (e-1) / -f
    +=改为更直观的表达方式
    式子变为 a = b = b + c - d - (e - 1) / f
    赋值号拆开，变为多个赋值式子
    b = b + c - d - (e - 1) / f
    a = b

不好的表达式的例子
    如何分析下面这个表达式？
    a = 5;
    c = (b=a+2) - (a=2);

    根据优先级，左边括号的赋值会先算
    即：b=a+2=5+2=7
    然后再算右边括号
    即：a=2
    最后算整个式子
    c=b-a=7-2=5
    综上所述，最终答案是
    a=2,b=7,c=5
